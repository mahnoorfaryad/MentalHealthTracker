# Mental Health Tracker

A Python + SQL project to track mental health data.

## Features
- User login
- Questionnaire input
- Lifestyle data entry
- Mood prediction
- Dashboard

## Tech Used
- Python (Tkinter / GUI)
- SQL Server / SQLite
- Python libraries: pandas, matplotlib, etc.

## How to Run
1. Run the Python script
2. Connect to SQL database
3. Follow instructions
